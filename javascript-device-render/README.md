# MailSlurp Device Rendering Examples

This folder contains native-target device render examples using the MailSlurp JavaScript SDK.

Run with:

```shell
npm install
API_KEY=your-api-key npm test
```

The test shows how to list native device targets, create renders from stored/imported email sources, create a render email address, wait for runs, list runs, view results, collect screenshot URLs, download images, and create public share links.
